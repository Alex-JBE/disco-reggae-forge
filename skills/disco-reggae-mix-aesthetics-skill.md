# disco-reggae-mix-aesthetics-skill

## Purpose
Use this skill to shape the final sonic finish of disco-reggae outputs.

This module covers:
- bass and kick balance
- width versus center
- dub depth
- sheen versus warmth
- vocal placement
- analog softness versus polished clarity

## When to use
Use this skill when:
- the output needs mix identity
- the track feels too dry, too flat, too bright, too muddy, or too modern-clean
- you need to choose between roots warmth, disco shine, or dub depth

## Workflow
1. Define low-end policy.
   - warm and round
   - punchy and danceable
   - dub-heavy
   - soft but deep

2. Define top-end policy.
   - silky
   - bright shimmer
   - restrained
   - tape-softened

3. Define space policy.
   - dry dance mix
   - moderate room and plate
   - dub send depth
   - selective echo throws

4. Define finish.
   - analog warm
   - polished retro
   - dub-smoked
   - clean sunset gloss

## If/Then rules
- If the bass loses definition, simplify kick/bass overlap.
- If the mix loses dance readability, reduce excessive dub haze.
- If the track feels too clean, add warmth or saturation.
- If the track feels too muddy, clear lower mids before thinning the bass.
- If disco lift disappears, restore top-end motion and chord clarity.

## Guardrails
- Avoid harsh top-end sparkle.
- Avoid overwashed dub FX across the whole track.
- Avoid sterile pop mastering as a default.
- Avoid underpowered bass in a groove-led track.

## Output format
Return:
- Low-end policy
- Top-end policy
- Space policy
- Finish
- Vocal placement
- Red flags
