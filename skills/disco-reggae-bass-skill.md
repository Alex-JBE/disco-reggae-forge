# disco-reggae-bass-skill

## Purpose
Use this skill to define the bass as the central glue between disco propulsion and reggae pocket.

In this hybrid, bass is not just support. It often decides:
- whether the groove feels grounded
- whether the reggae side remains believable
- whether disco motion feels embodied rather than cosmetic
- whether the entire track breathes or stiffens

## When to use
Use this skill when:
- the bassline is underdefined
- the track feels too top-heavy
- the groove lacks weight or authority
- the user wants more bounce, more roots depth, more dancefloor motion, or more dub presence

## Workflow
1. Choose bass center.
   - roots-rounded
   - disco-driving
   - boogie-funked
   - dub-heavy
   - lovers-soft
   - club-tight

2. Choose note behavior.
   - sustained
   - walking-short
   - syncopated pockets
   - octave bounce
   - rolling anchor
   - phrase-led hook

3. Define rhythmic relationship with the kick.
   - locked
   - interlocked
   - bass-leading
   - kick-leading
   - elastic conversation

4. Choose tonal behavior.
   - warm and woody
   - rubbery
   - soft-deep
   - clean round
   - saturated
   - dub-smoked

5. Set movement density.
   - sparse
   - moderate
   - active
   - hook-dominant

## If/Then rules
- If the track feels too disco-glossy, make the bass more pocket-conscious and less decorative.
- If the track feels too roots-static, give the bass more forward dance motion.
- If the groove feels stiff, change bass phrasing before adding percussion.
- If the bass is too busy, remove notes and preserve contour.
- If the chorus lacks lift, use octave or phrase expansion rather than making the bass louder.
- If the dub factor rises, let sustain and space matter more.

## Bass archetypes
- warm roots anchor
- rolling dancefloor spine
- lovers glide bass
- boogie-reggae bounce
- dub pressure bass
- club disco-reggae low-end

## Guardrails
- Avoid generic funk bass that ignores reggae pocket.
- Avoid sub-only bass with no phrase identity.
- Avoid over-filling every bar.
- Avoid EDM-style low-end design that erases the hybrid.
- Avoid thin bass in a groove-centered style.

## Output format
Return:
- Bass center
- Note behavior
- Kick relationship
- Tonal behavior
- Density level
- Hook role
- Red flags
