# disco-reggae-drums-skill

## Purpose
Use this skill to define the drum identity of the disco-reggae hybrid.

This module covers:
- kick stance
- snare/rim placement
- hi-hat behavior
- percussion support
- disco pulse versus reggae pocket
- live versus polished feel

## When to use
Use this skill when:
- the drums feel stylistically confused
- the track needs more dance motion or more pocket
- the user wants a more rootsy, glossier, clubbier, dubbier, or breezier rhythmic bed

## Workflow
1. Choose kick behavior.
   - four-on-floor
   - disco-light pulse
   - hybrid pulse
   - one-drop influenced
   - bass-supportive sparse kick

2. Choose backbeat identity.
   - rim-led
   - snare-led
   - clap support
   - dry hit
   - soft layered backbeat

3. Define hat and shaker motion.
   - straight shimmer
   - disco hiss
   - loose shaker roll
   - airy top motion
   - restrained top-end

4. Define percussion role.
   - minimal
   - hand percussion support
   - conga/bongo lift
   - tambourine lift
   - dub accent percussion

## If/Then rules
- If disco is leading, strengthen pulse and shimmering motion.
- If reggae is leading, keep the pocket elastic and bass-aware.
- If the beat feels stiff, loosen top-end phrasing or rim placement.
- If the beat feels too roots-static, add dance momentum.
- If the beat feels too generic disco, reintroduce skank-aware space and bass conversation.

## Guardrails
- Avoid generic tropical percussion overload.
- Avoid flattening reggae nuance into straight pop drums.
- Avoid over-bright hats masking the pocket.
- Avoid too many fills.

## Output format
Return:
- Kick behavior
- Backbeat identity
- Top-end motion
- Percussion role
- Pocket level
- Red flags
