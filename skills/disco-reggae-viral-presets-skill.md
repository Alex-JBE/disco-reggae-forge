# disco-reggae-viral-presets-skill

## Purpose
Use this skill to create clip-ready disco-reggae moments that feel instantly readable without flattening the style into trend bait.

## When to use
Use this skill when:
- the user wants short-form hook presets
- the output is intended for clips, teasers, edits, reels, or loop previews
- the hybrid needs immediate recognizability

## Workflow
1. Identify the instant-read element.
   - bass hook
   - vocal phrase
   - skank-and-kick lift
   - chorus burst
   - percussion break
   - dub throw moment

2. Shorten time to identity.
3. Choose replay mechanic.
4. Preserve groove and pocket.
5. Add one memorable contrast cue.

## If/Then rules
- If the clip feels too generic, strengthen hybrid-specific groove details.
- If the hook arrives too late, move the identity marker forward.
- If the clip feels too pop-only, restore bass and offbeat logic.
- If the result loses warmth, reduce trend-edit aggression.

## Guardrails
- Avoid reducing the style to touristy sunshine clichés.
- Avoid making every clip chorus-only.
- Avoid trend-template sameness.
- Avoid sacrificing groove for gimmick edits.

## Output format
Return:
- Clip identity
- Instant-read element
- Replay mechanic
- Groove anchor
- Contrast cue
- Red flags
