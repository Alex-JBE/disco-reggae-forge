# disco-reggae-gmiv-presets-skill

## Purpose
Use this skill to generate compact, reusable prompts through four linked axes:
- Genre
- Mood
- Imagery
- Velocity

## When to use
Use this skill when:
- the interface needs compact preset cards
- the user wants fast generation seeds
- emotional language needs to be translated into production-aware guidance

## Workflow
1. Set Genre branch.
2. Set Mood.
3. Set Imagery.
4. Set Velocity.
5. Translate all four into production behavior.

## Example preset
Genre: lovers disco reggae  
Mood: warm romantic glide  
Imagery: mirror-ball light over humid midnight air  
Velocity: medium rolling sway with soft dancefloor lift

## If/Then rules
- If imagery is too generic, make it more tactile and scene-specific.
- If mood clashes with the branch, either shift the branch or narrow the mood.
- If velocity feels mismatched, describe perceived movement, not just BPM.
- If the preset sounds pretty but unusable, add bass/groove translation.

## Guardrails
- Avoid vague mood poetry with no sonic consequence.
- Avoid generic tropical imagery.
- Avoid BPM-only thinking.
- Avoid presets that do not imply arrangement or groove behavior.

## Output format
Return:
- Genre
- Mood
- Imagery
- Velocity
- Production translation
- Red flags
