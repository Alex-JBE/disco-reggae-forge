# disco-reggae-skank-keys-skill

## Purpose
Use this skill to control the offbeat support system that gives the hybrid its reggae credibility and rhythmic sparkle.

This module covers:
- skank guitar behavior
- bubble organ/keys logic
- chord stab shape
- offbeat density
- interaction with disco comping
- hybrid harmonic rhythm

## When to use
Use this skill when:
- the track lacks offbeat identity
- the user wants more reggae pocket
- the chords feel too static or too obvious
- the hybrid needs rhythmic support beyond bass and drums

## Workflow
1. Choose dominant support instrument.
   - muted guitar skank
   - bubble organ
   - bright electric piano
   - clav-like stab
   - hybrid keys and guitar
   - synth-skank adaptation

2. Choose attack behavior.
   - dry and clipped
   - soft and brushed
   - bright and percussive
   - dub-muted
   - silky disco accent

3. Define rhythmic behavior.
   - strict offbeat
   - offbeat plus anticipation
   - bubble pattern
   - hybrid syncopation
   - sparse punctuation

4. Define chord density.
   - dyads
   - triads
   - extensions
   - suspended color
   - short voicings only

5. Set relationship to disco harmony.
   - pure reggae support
   - disco-bright hybrid
   - boogie-chord crossover
   - dub-minimal support

## If/Then rules
- If the track feels too straight, increase offbeat definition.
- If the track feels too busy, reduce simultaneous guitar and key chatter.
- If disco chords are dominating, restore clipped offbeat logic.
- If the result becomes too roots-traditional, brighten voicings or comping motion slightly.
- If the branch is dub-heavy, allow more subtraction and echo.
- If the branch is lovers-oriented, soften the attacks and sweeten voicings.

## Guardrails
- Avoid constant strumming.
- Avoid piano-house comping that erases reggae logic.
- Avoid overextended jazz voicings that clog the groove.
- Avoid making the offbeat support louder than the bass center.
- Avoid novelty skank stereotypes.

## Output format
Return:
- Dominant support instrument
- Attack behavior
- Rhythmic behavior
- Chord density
- Hybrid relation
- Red flags
