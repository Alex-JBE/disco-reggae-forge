# disco-reggae-harmony-color-skill

## Purpose
Use this skill to shape the emotional and tonal color of the hybrid through chord choices, voicing behavior, and harmonic pacing.

This module covers:
- major/minor blend
- sweetness versus earthiness
- soulful color
- sunset warmth
- dub ambiguity
- harmonic motion rate

## When to use
Use this skill when:
- the harmony feels emotionally generic
- the user wants warmer, sweeter, deeper, duskier, or brighter color
- the branch needs stronger tonal distinction
- the track needs a clearer emotional world

## Workflow
1. Choose tonal center.
   - major-sunlit
   - minor-warm
   - bittersweet
   - soulful-major/minor blend
   - dub-ambiguous
   - devotional uplift

2. Choose voicing color.
   - simple triads
   - sevenths
   - sixths and ninths
   - suspended voicings
   - organ-thick color
   - bright electric keys

3. Choose harmonic motion.
   - static vamp
   - slow cycle
   - medium dance cycle
   - chorus lift progression
   - dub-minimal drift

4. Choose emotional color.
   - communal joy
   - lovers tenderness
   - smoky late-night
   - sunset glow
   - heady introspection
   - celebratory uplift

## If/Then rules
- If the track feels too cheerful and thin, add depth through bass-led minor warmth.
- If the track feels too dark and heavy, introduce brighter upper harmony or sweeter voicings.
- If the branch is lovers-oriented, use more emotional clarity and softness.
- If the branch is dub-heavy, let ambiguity and repetition do more work.
- If the branch is disco-led, keep harmony moving enough to sustain lift.
- If the chords become too busy, reduce changes and deepen the groove instead.

## Guardrails
- Avoid generic beach-pop major harmony.
- Avoid overly complex jazz harmony that steals rhythmic attention.
- Avoid constant uplift with no shadow.
- Avoid total harmonic stasis unless the branch explicitly wants deep dub hypnosis.

## Output format
Return:
- Tonal center
- Voicing color
- Harmonic motion
- Emotional color
- Branch fit
- Red flags
