# disco-reggae-dub-space-skill

## Purpose
Use this skill to define how much dub language enters the disco-reggae hybrid and how space behaves as an active musical force.

This module covers:
- delay behavior
- reverb philosophy
- drop-out tactics
- send-return emphasis
- spatial contrast
- how absence creates groove tension

## When to use
Use this skill when:
- the user wants more dub atmosphere
- the track feels too dry or too literal
- the arrangement needs more breathing space
- the hybrid needs a stronger sense of depth and movement

## Workflow
1. Choose dub intensity.
   - subtle
   - moderate
   - pronounced
   - near-dub-version

2. Choose echo behavior.
   - vocal throws
   - snare/rim throws
   - skank repeats
   - chord haze
   - full send-return punctuation

3. Choose reverb role.
   - room glue
   - plate sweetness
   - spring color
   - deep chamber
   - selective abyss

4. Choose subtraction style.
   - occasional mutes
   - bass spotlight
   - drum strip-down
   - dub breakdown
   - FX-led pocket suspension

5. Set clarity protection.
   - dancefloor clarity first
   - balanced
   - space-first with rhythmic discipline

## If/Then rules
- If the track is dancefloor-led, keep dub FX selective and groove-safe.
- If the branch is club dub disco, let strip-outs and sends do more structural work.
- If the result feels washed out, reduce tail length before reducing depth entirely.
- If the track feels flat, create contrast through selective emptiness.
- If the groove loses punch, pull key anchors back to the center and dry plane.
- If the vocal needs memorability, use echo punctuation rather than constant wetness.

## Dub-space archetypes
- sweet plate glow
- echo-punctuated groove
- bass-first dub chamber
- selective disco dub
- strip-down and return
- smoked sunset depth

## Guardrails
- Avoid drenching the whole mix.
- Avoid making every phrase an echo event.
- Avoid dub clichés with no structural purpose.
- Avoid blurring the kick-bass relationship.
- Avoid ambient wash replacing groove.

## Output format
Return:
- Dub intensity
- Echo behavior
- Reverb role
- Subtraction style
- Clarity protection
- Red flags
