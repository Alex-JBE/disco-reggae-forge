# disco-reggae-inspire-skill

## Purpose
Use this skill to generate new directions when the current disco-reggae idea feels too obvious, too soft, too retro, or too generic.

This module should produce:
- hybrid pivots
- alternate emotional frames
- rhythm-first variations
- dub-space reinterpretations
- crossover prompts
- tighter branch-specific directions

## When to use
Use this skill when:
- the user is stuck
- the current idea lacks personality
- the hybrid needs a sharper identity
- you want variation without losing the groove center

## Workflow
1. Identify current center.
2. Choose pivot type.
   - rhythmic
   - harmonic
   - vocal
   - dub-space
   - retro-futurist
   - branch crossover

3. Generate 3 to 7 inspiration paths.
4. Keep one safe, one moderate, and one adventurous option.
5. Translate the strongest option into a production-facing prompt.

## If/Then rules
- If the current idea is too sweet, add bass seriousness or dub tension.
- If it is too roots-static, add disco lift or boogie polish.
- If it is too glossy, restore human pocket.
- If it is too generic, narrow the branch and imagery.
- If it becomes random fusion, re-anchor bass and groove.

## Example pivots
- What if lovers disco reggae had deeper dub strip-outs between choruses?
- What if cosmic disco reggae kept roots bass seriousness?
- What if afro-disco reggae used more hand percussion but less tourist-pop brightness?
- What if club dub disco stretched the groove into a DJ-friendly 12-inch form?
- What if boogie reggae emphasized talkbox-like synth doubles against skank guitar?

## Guardrails
- Avoid gimmick fusion.
- Avoid novelty “beach party” logic.
- Avoid vague inspiration with no production consequence.
- Avoid crossover ideas that erase the pocket.

## Output format
Return:
- Current center
- 3 to 7 inspiration directions
- Best direction
- Why it works
- Production-facing translation
