# disco-reggae-vocal-presets-skill

## Purpose
Use this skill to shape how vocals function in disco-reggae.

Possible vocal roles:
- sweet lead
- chant hook
- communal chorus
- toasting accent
- romantic glide
- dub fragment
- soulful uplift

## When to use
Use this skill when:
- the output includes vocals
- the user wants stronger hooks, harmony, chants, or lovers-style sweetness
- the vocal needs to sit correctly between disco brightness and reggae pocket

## Workflow
1. Choose vocal role.
2. Choose phrasing style.
3. Choose harmony density.
4. Choose processing level.
5. Define lyrical tone.

## If/Then rules
- If the branch is lovers-oriented, prioritize sweetness and melodic grace.
- If the branch is dub-heavy, allow fragments, echoes, and selective lyric sparseness.
- If the branch is dancefloor-facing, use clearer hooks.
- If the vocal is overpowering the groove, reduce density and leave more phrase space.
- If the result feels emotionally flat, strengthen harmony or chorus memory.

## Guardrails
- Avoid generic vacation-pop vocals.
- Avoid over-singing against the groove.
- Avoid excessive melisma unless stylistically necessary.
- Avoid crowd-chant clichés in every track.

## Output format
Return:
- Vocal role
- Phrasing style
- Harmony density
- Processing level
- Lyrical tone
- What to avoid
