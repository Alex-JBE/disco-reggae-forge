# disco-reggae-subgenres-skill

## Purpose
Use this skill to sort disco-reggae ideas into distinct hybrid lanes instead of one vague blended style.

This module handles:
- subgenre naming
- branch identity
- hybrid coherence
- triplet logic
- neighboring branches
- stylistic guardrails

## When to use
Use this skill when:
- the user selects a disco-reggae branch
- you need to generate subgenre-specific prompts
- one hybrid idea needs stylistic sharpening
- the app needs clickable subgenre identities

## Example identity summaries
- roots disco reggae: grounded bass, human pocket, disco lift with roots restraint
- lovers disco reggae: sweet vocal warmth, romantic glide, clean dance motion
- dub disco reggae: deeper space, echo drops, bass-weighted hypnosis
- cosmic disco reggae: wider synths, starry pads, smoother disco thrust
- boogie reggae: tighter bass funk, polished rhythm guitars/keys, bright late-night warmth
- sunset reggae dance: breezy, open, melodic, communal, coastal
- club dub disco: DJ-minded groove, extended pockets, dub FX discipline

## Workflow
1. Identify the branch.
2. Define dominant rhythmic behavior.
3. Define bass and chord role.
4. Define vocal priority.
5. Define amount of dub space.
6. Define production polish level.
7. State adjacent branches and red flags.

## If/Then rules
- If the branch is roots-based, reduce synthetic gloss.
- If the branch is cosmic or synth-led, preserve pocket so it still feels reggae-aware.
- If the branch is dub-heavy, let space and subtraction work harder than melody.
- If the branch is lovers-oriented, prioritize vocal warmth and sweetness over aggression.
- If the branch becomes generic beach-pop, restore bass seriousness and rhythmic identity.
- If the branch becomes too retro-disco only, restore skank, bubble, or dub cues.

## Guardrails
- Do not collapse all branches into “funky reggae”.
- Avoid stereotypes as subgenre markers.
- Avoid tropical-pop flattening.
- Avoid overloading every branch with the same horns, claps, and bright guitars.

## Output format
Return:
- Subgenre branch
- Identity summary
- Rhythmic behavior
- Bass/chord relation
- Vocal stance
- Space policy
- Neighbor branches
- Red flags
