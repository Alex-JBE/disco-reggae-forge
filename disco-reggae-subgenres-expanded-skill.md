# disco-reggae-subgenres-expanded-skill

## Purpose
Use this skill to move beyond broad hybrid labels and route prompts into more exact disco-reggae branches with sharper production outcomes.

This module expands the first-wave subgenre map and provides:
- deeper branch resolution
- branch-specific production cues
- hybrid routing logic
- neighboring-branch awareness
- identity boundaries

## When to use
Use this skill when:
- the broad branch label is not specific enough
- the user wants tighter stylistic targeting
- multiple disco-reggae directions need differentiation
- the product needs richer subgenre routing

## Branch routing workflow
1. Identify whether the user wants roots, romance, dub, polish, breeziness, rhythm, or crossover energy.
2. Narrow to one of the expanded branches.
3. Define bass seriousness.
4. Define offbeat support level.
5. Define disco gloss level.
6. Define dub-space allowance.
7. Define vocal centrality.
8. Set production-era feel.

## If/Then rules
- If the user wants warmth and sweetness, route toward lovers or soft-sway branches.
- If the user wants heavier groove and deeper space, route toward dub branches.
- If the user wants cleaner polish and late-night shine, route toward boogie/synth branches.
- If the user wants more organic movement, route toward rhythm-forward branches.
- If the user asks for crossover accessibility, use crossover branches but preserve bass and offbeat logic.
- If the result becomes too generic, narrow branch language and era feel.

## Production-era feel options
- late 70s hybrid warmth
- early 80s boogie sheen
- dub-console analog depth
- modern retro polish
- DJ re-edit extension feel

## Guardrails
- Do not let all expanded branches sound like the same song with different adjectives.
- Avoid crossover branches turning into generic radio-pop.
- Avoid roots branches losing dance readability.
- Avoid polished branches becoming sterile.
- Avoid breezy branches becoming tourist music.

## Output format
Return:
- Expanded branch
- Branch family
- Production logic
- Bass seriousness
- Offbeat support level
- Dub-space level
- Vocal centrality
- Era feel
- Red flags
